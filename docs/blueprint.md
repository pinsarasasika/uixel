# **App Name**: UIXEL Digital

## Core Features:

- Animated Glass Cards: Implement glassmorphism style cards with subtle animations to present key content.
- Portfolio Filtering: Enable filtering portfolio projects by category: Web, UI/UX, AI, Branding.
- Pre-made Templates Showcase: Offer website templates with live preview, pricing, and purchase options.
- SEO Optimization Tool: Use AI to generate effective titles and descriptions, acting as a tool for creating optimized content.
- Contact Form with Validation: Provide a contact form with real-time validation and feedback for users.
- Smooth Scroll Animations: Add subtle, smooth scroll animations to enhance the user experience.
- Dark/Light Mode Toggle: Include a toggle to switch between dark and light themes for accessibility and preference.

## Style Guidelines:

- Primary color: Soft purple (#A899FF) for a modern, innovative feel.
- Background color: Dark navy blue (#1A1E27) to enhance glassmorphism and neon accents.
- Accent color: Electric blue (#7DF9FF) for CTAs and interactive elements.
- Body and headline font: 'Inter' sans-serif font for a clean, futuristic aesthetic.
- Use minimalist line icons with subtle glow effects.
- Implement a grid-based layout with floating glass cards to highlight content.
- Incorporate subtle motion and micro-interactions to enhance user engagement.
